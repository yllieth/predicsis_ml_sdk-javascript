  • create(params)

    Send POST request to the modalities_sets API resource.

    You can / must give the following parameters to ask for a modalities set creation: 

     {
   variable_id: "5329601c1757f446e6000002"
   dataset_id:  "53c7dea470632d3417020000",
 }

    Both variable_id and dataset_id are required.

    Parameters
    ParamTypeDetails
    paramsObject

    See above example.

    Returns
    Promise

    Returned modalities set does not contain modalities themselves. If you want them, you must explicitly get them. In fact, you will get an object like: 

      {
    id": "53fdfa7070632d0fc5030000",
    created_at: "2014-05-02T17:13:56.687Z",
    user_id: "5363b25c687964476d000000",
    dataset_id: "53c7dea470632d3417020000",
    variable_id: "5329601c1757f446e6000002",
    job_ids: [ "53c8c88970632d3b9a030001" ]
  }
