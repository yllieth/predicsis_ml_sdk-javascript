learning_dataset_id : is the id of the parent dataset. During the process this parent dataset will be splitted into 2 subsets (train and test). This field in project resource is designed to store the parent dataset.

reports_ids : is an array of ids where reports should always be stored in the same order:

  1. classifier evaluation report for train part
  2. classifier evaluation report for test part
  3. univariate supervised report
  4. univariate unsupervised report
