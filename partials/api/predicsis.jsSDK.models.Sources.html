Sources
service in module predicsis.jsSDK.models

Description

Sources are a representation of an uploaded file on our storage. At time, all uploads are sent to Amazon S3.

post /sources Sources.create()
get /sources Sources.all()
get /sources/:id Sources.get()
patch /sources/:id Sources.update() async
delete /sources/:id Sources.delete()
Official documentation is available at https://developer.predicsis.com/doc/v1/data_management/source/

  • create(params)

    Send POST request to the source API resource.

    This request is going to create and persist in database a source object regarding to a given uploaded file. So, you must upload a file first.

    You can / must give the following parameters to ask for a source creation: 

     {
   name: "Source of dataset.csv"
   key:  "path/to/my/file/on/s3/source.csv",
 }

    Both name and key are required.

    ParamTypeDetails
