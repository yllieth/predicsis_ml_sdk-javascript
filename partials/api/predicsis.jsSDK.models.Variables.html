Variables
service in module predicsis.jsSDK.models

Description

get /dictionary/:dictionaryId/variables Variables.all()
get /dictionary/:dictionaryId/variables/:variableId Variables.get()
patch /dictionary/:dictionaryId/variables/:variableId Variables.update()
Official documentation is available at https://developer.predicsis.com/doc/v1/dictionary/variable/

Output example: 

{
id: "5492e2a6776f720001000500",
created_at: "2014-12-18T14:20:22.858Z",
updated_at: "2014-12-18T14:20:22.858Z",
name: "Sepal Length",
type: "continuous",
use: true,
description: null,
dictionary_id: "5492e2b1617070000b1d0000",
modalities_set_ids: []
}

As a variable cannot live without being attached to a dictionary, all request need a dictionaryId.
