  • create(params)

    Send POST request to the dictionary API resource. This request is going to generate a dictionary regarding to a given dataset. This generation is delegated to ML core tool. That's why this request is asynchronous.

    You can give the following parameters to ask for dictionary generation: 

    {
  dataset_id: "53c7dea470632d3417020000",
  name:       "Dictionary of my awesome dataset"
}
    Parameters
    ParamTypeDetails
    paramsObject

    See above example.

    Returns
    Object

    Promise of a new dictionary

  • createFromDataset(dataset)
