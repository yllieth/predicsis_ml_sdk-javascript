  • update(id, changes)

    Update specified user You can update the following parameters:

    • {String} name
    • {String} email
    • {String} current_password
    • {String} password


    Note: If changes param contains both current_password and password, these two properties are sent to PATCH /users/update_password. Any other param is sent to PATCH /users/:id.

    Parameters
    ParamTypeDetails
    idString

    Id of the user you want to update

    changesObject

    see above description to know parameters you are able to update

    Returns
    Promise

    Updated user and/or password
